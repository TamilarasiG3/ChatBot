# Chatbot (HTML, CSS, JavaScript)

A simple and responsive manual chatbot web application built using HTML, CSS, and JavaScript. This chatbot provides predefined responses based on user input and is perfect for beginners learning frontend development.

## 🚀 Features
💡 Predefined chatbot responses
🎨 Clean and modern UI
🌙 Dark theme design
⚡ Fast and lightweight (no backend required)
📱 Fully responsive (works on mobile & desktop)
🔘 Send message using button or Enter key

## ⚙️ How It Works
User types a message in the input box
JavaScript checks the input
Matches it with predefined responses
Displays the bot reply instantly

## 🔧 Customization

You can edit responses in script.js:

const botResponses = {
  hello: "Hello! How can I help you?",
  hi: "Hi there!",
  bye: "Goodbye!"
};
